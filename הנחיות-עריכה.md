## 💻 נוהל עבודה מקומית (למתקדמים / בלי אינטרנט לפאנל)

1. טרמינל בתיקיית `pro-law-connect`
2. פעם ראשונה בלבד: `npm install`
3. שני טרמינלים: `npm run dev` (האתר) + `npx decap-server` (מנוע הפאנל המקומי)
4. אתר: http://localhost:8080 · פאנל בלי התחברות: http://localhost:8080/admin/
5. בסיום — להעלות ל-GitHub: `git add -A` → `git commit -m "תיאור"` → `git push`
